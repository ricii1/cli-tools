# cli-tools
bash script for command-command to simplify remembering task

## Quick Setup
1. Clone this repo
    ```sh
    git clone https://github.com/ricii1/cli-tools.git
    chmod +x cli-tools/bin/*
    ```
2. For zsh
    ```
    echo 'export PATH="$HOME/cli-tools/bin:$PATH"' >> ~/.zshrc
    source ~/.zshrc
    ```
    For bash
    ```
    echo 'export PATH="$HOME/cli-tools/bin:$PATH"' >> ~/.bashrc
    source ~/.bashrc
    ```
